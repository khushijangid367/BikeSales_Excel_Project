# BikeSales_Excel_Project

## 📌 Overview
This project analyzes bike sales data using Microsoft Excel to uncover insights about customer demographics, purchasing behavior, and sales trends. The goal is to transform raw data into meaningful visualizations and actionable insights.

---

## 🎯 Objectives
- Understand customer buying patterns
- Analyze factors influencing bike purchases
- Create interactive dashboards
- Improve data cleaning and visualization skills in Excel

---

## 📊 Tools Used
- Microsoft Excel
  - Pivot Tables
  - Charts & Graphs
  - Data Cleaning
  - Dashboard Creation

---

## 📁 Dataset
The dataset includes:
- Customer demographics (Age, Gender, Income)
- Commute distance
- Region
- Purchase status (Bought bike or not)

---

## 🔧 Data Cleaning Steps
- Removed duplicates
- Handled missing values
- Standardized categorical data
- Created new calculated columns (if applicable)

---

## 📈 Analysis Performed
- Income vs Purchase behavior
- Age group analysis
- Gender-based purchasing trends
- Commute distance impact on bike buying

---

## 📊 Dashboard Features
- Interactive filters (slicers)
- Visual representation of key metrics
- Easy-to-understand charts for decision making

---

## 💡 Key Insights
- Higher income groups tend to purchase more bikes
- Middle-aged customers show higher buying interest
- Commute distance affects purchase decisions
- Gender differences observed in buying patterns
